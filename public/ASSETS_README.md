# Required Assets for LawAI

Please add the following files to the `public` folder:

## Favicon & Icons

1. **favicon.ico** - 32x32 or 48x48 pixels, ICO format
   - This is the main favicon shown in browser tabs

2. **icon-192.png** - 192x192 pixels, PNG format
   - Used for PWA and Android devices

3. **icon-512.png** - 512x512 pixels, PNG format
   - Used for PWA splash screens

4. **apple-touch-icon.png** - 180x180 pixels, PNG format
   - Used when users add the site to their iOS home screen

## Social Media / OG Image

5. **og-image.png** - 1200x630 pixels, PNG format
   - This image appears when the site is shared on social media
   - Should include:
     - LawAI logo
     - Tagline: "AI-Powered Legal Assistant"
     - Clean, professional design
     - Brand colors (black/white theme)

## Logo

6. **logo.png** - Your main logo, recommended 200x200 pixels minimum
   - Used in structured data and various places
